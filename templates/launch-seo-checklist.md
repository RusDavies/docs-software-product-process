# Launch SEO Checklist

## Scope

Release or web property being checked.

## Pre-Launch

- [ ] SEO plan exists where required
- [ ] page inventory complete
- [ ] metadata plan complete
- [ ] sitemap generated
- [ ] robots.txt reviewed
- [ ] canonical behavior reviewed
- [ ] structured data validated where used
- [ ] share previews checked
- [ ] analytics/search measurement configured or intentionally deferred
- [ ] staging/private indexing protections checked

## Launch Verification

- [ ] production pages return correct status codes
- [ ] priority pages render important content
- [ ] priority pages are indexable
- [ ] sitemap reachable
- [ ] robots.txt reachable
- [ ] redirects work
- [ ] 404 page works
- [ ] mobile smoke check passed
- [ ] performance smoke check passed or exception approved
