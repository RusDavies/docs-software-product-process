# Audit Readiness Checklist

## Target

- SOC 2 Type I:
- SOC 2 Type II:
- ISO/IEC 27001 alignment:
- ISO/IEC 27001 certification:
- Enterprise security questionnaire readiness:

## Scope

- [ ] System scope defined
- [ ] Environment scope defined
- [ ] Data scope defined
- [ ] Vendor/dependency scope defined
- [ ] Exclusions documented

## Controls and Evidence

- [ ] Control register exists
- [ ] Evidence register exists
- [ ] Control owners assigned
- [ ] Evidence owners assigned
- [ ] Evidence retention expectations defined
- [ ] Gaps tracked in backlog

## Policies / Procedures

- [ ] Policy index exists
- [ ] Access control documented
- [ ] Secure SDLC documented
- [ ] Change management documented
- [ ] Incident response documented
- [ ] Vendor management documented
- [ ] Risk management documented
- [ ] Backup/recovery documented
- [ ] Logging/monitoring documented

## Operating Evidence

- [ ] Access review evidence available
- [ ] Change/deployment evidence available
- [ ] Vulnerability management evidence available
- [ ] Incident response test or records available
- [ ] Backup/restore evidence available
- [ ] Vendor review evidence available
- [ ] Risk assessment evidence available

## Customer-Facing Readiness

- [ ] Security overview prepared
- [ ] Compliance status is accurate
- [ ] Questionnaire answers prepared
- [ ] Subprocessor/vendor list prepared where applicable
- [ ] Security contact / disclosure path defined
