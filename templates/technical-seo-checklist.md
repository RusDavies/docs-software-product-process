# Technical SEO Checklist

## Crawl / Index

- [ ] robots.txt correct
- [ ] sitemap exists and validates
- [ ] canonical URLs correct
- [ ] intended pages indexable
- [ ] private/staging pages noindexed or blocked
- [ ] redirects intentional
- [ ] 404/410 behavior correct

## Page Structure

- [ ] semantic HTML used
- [ ] titles present
- [ ] descriptions present where needed
- [ ] heading hierarchy sensible
- [ ] internal links crawlable
- [ ] anchor text descriptive
- [ ] images have useful alt text where appropriate

## Structured Data / Social

- [ ] structured data validates where used
- [ ] Open Graph previews work
- [ ] Twitter/X card previews work where relevant

## Performance / Mobile

- [ ] mobile rendering works
- [ ] LCP acceptable or exception approved
- [ ] INP acceptable or exception approved
- [ ] CLS acceptable or exception approved
- [ ] images/fonts/scripts optimized enough for release
